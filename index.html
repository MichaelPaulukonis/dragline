D R A G L I N E

  • Click and drag to reposition text block on the grid.
  • Release the mouse to drop the text block in its new position.
  • UP/DOWN arrows to move the selected text block up or down in the display order (higher blocks have display priority).
  • RIGHT/LEFT arrow to add/remove a text block
  • SPACE - cycle through various fill-characters
  • r - reset with new blocks and new locations
  • n - load new blocks from Tumblr (!!!)
  • i to show/hide this info box

Check out the code at github.com/MichaelPaulukonis/dragline.git.
