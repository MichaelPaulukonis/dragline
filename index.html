D R A G L I N E

a different kind of text landscape

  • Mouse: Click and drag to move text blocks.
  • Arrow Keys: Move selected block.
  • SHIFT + Up/Down: Change block z-index (move up or down).
  • SHIFT + Left/Right: Cycle through blocks.
  • SPACE: Cycle through fill characters.
  • r: Reset text areas.
  • SHIFT + S: Save a black-and-white copy of the canvas as dragline.YYYYMMDD.hhmmss.png.
  • n: Fetch new blocks from Tumblr.
  • i or ESC: Toggle this info box.

Check out the code at github.com/MichaelPaulukonis/dragline.git.
